## 软件 mizan(点货软件)
- 自营 **点货 App**,已上架 **安卓 + 苹果**。
- **超十万 SKU**,基本涵盖华人商超店面所需产品 —— **唯独没有食品类**(重要:别在文案里承诺食品/调料/康师傅这类,平台没有)。

## 客户画像与运营事实(2026-06-28 用户口述·选题/钩子/真实故事的事实源)

### 1. 现有下单客户主要国家
西班牙 / 意大利 / 葡萄牙 / 智利 / 阿根廷 / 希腊 / 墨西哥 / 厄瓜多尔 / 肯尼亚 / 南非。
**欧洲占比最高,尤其西班牙。**
⚠️ 注意:有客户 ≠ 有仓。仅 **西/葡/智/墨** 有海外仓;意/阿/希腊/厄/肯/南非有客户但**无仓**(话术只讲中国站直发)。

### 2. 客户开什么店
海外华人零售商为主,**百元店 / 百货店**为主;店铺 **500㎡–5000㎡**;部分多门店;少量批发商/区域供货商(占比低)。

### 3. 下单最多最稳最复购的客户(核心买单人群)
**经营多年、有实体门店的华人零售商**——尤其 **500㎡以上百货店 / 连锁百货 / 百元店 / 华人商超**。
每月补货、采购周期稳、复购高(卖的是日用品=持续消耗型)。→ 选题主攻这类人。

### 4. 卖得最好的品类(选品清单选题真实依据)
- **欧洲**:家居百货 / 厨房用品 / 收纳用品 / 家居装饰 / 手动五金工具 / 汽车用品 / 季节性产品。
- **南美**:电动工具 / 五金工具 / 美妆用品 / 服装服饰 / 箱包系列 / 家居用品。
- **整体核心**:日用品 + 家居百货最稳。

### 5. 客户真实故事(身份代入型判断4的真料)
以前每年回中国采购:机票酒店贵 + 选品耗时。现在 Mizan 手机下单,一年省几次往返中国的时间和钱。
**中国直采落地成本比当地采购通常低 20%以上**(算账型判断1的硬数字)。

### 6. 客户最常夸 / 最常顾虑(钩子 + 异议处理)
**夸**:①产品丰富(一个平台10万+SKU,一站满足百货店大部分采购,不用到处找供应商)②价格透明(工厂出厂价,减中间环节)③采购方便(手机下单,不用频繁回国)④海外仓支持(部分商品海外仓发货,补货快)。
**顾虑(评论真实问题,要在内容里主动答)**:价格是真的吗?质量怎么样?能不能混装?运费怎么算?多久能到?有没有现货?海外仓有没有库存?
**最核心 = 第一次合作的信任问题**。破冰点:客户来义乌展厅,最大感受=「中国直采价格和当地差这么大」。→ **信任=最大转化杠杆,内容多用展厅实拍/真实价差/客户原话破信任。**

### 7. 获客渠道 + 最想要的客户
- **现有来源**:线上(抖音/视频号/小红书/华人社群/海外华商转介绍)+ 线下(展厅到访/海外仓客户推荐/老客转介绍)。
- **最想要**:①海外华人百货店老板(500㎡+、成熟门店、持续补货)②多门店连锁经营者 ③当地批发商(能覆盖更多终端)。

## 客户真实答复补充(2026-06-30 客户口述·可直接进内容·不编造)
> 这批解锁了**顾虑型(质量/运费/到货)+ 选品型 + 故事型(真人真事)**。口径仍按 [[mizan-5video-production-state]] 定的**第三人称专业洞察**(供应商/平台视角,不装店主)。

### A. 质量保障(答顾虑"质量怎么样?")
平台产品来自**合作工厂,工厂经质检**;到 mizan 仓库后**再抽检验货**,确保发出符合要求;客户收货有售后问题,**专门售后人员全程跟进**,**客户不用自己对接工厂**。

### B. 运费(答"运费怎么算?")
主走**海运**(空运成本太高,一般不选);**公开透明**,按**集装箱体积 + 目的地国家**计算,不同国家有别。⚠️**视频别讲太细,只突出"价格透明 / 没有隐形收费"**。

### C. 到货时间(答"多久能到?")
中国站发货到**欧洲/南美,海运约 2 个月左右**;具体视国家/港口/清关而定。**有海外仓→补货更快(就近补);无海外仓→按实际物流时效。**(与海外仓"10-15天补货"口径一致:那是海外仓补货,中国站直发≈2月。)

### D. App 库存
**暂不重点讲**(别在视频里强调库存)。

### ⛔ 真实料过滤铁律(2026-06-30 客户反馈·防误注入)
mizan **无食品**(26 类目全百货)。真实料来自"华人超市"评论,但华人超市卖食品/百货混着,所以评论里有「**过期 / 生产日期 / 保质期 / 食品变质 / 临期**」类吐槽——**这些对 mizan 完全不适用,绝不能注入 mizan 文案**(客户当面纠:"我们就没有食品,怎么会有过期")。系统在真实料注入前**必须按品牌事实过滤掉食品类痛点**。✅ 适用的真实顾虑=价格真假/质量/运费/到货/退换货(退换货客户确认"是有的",用新真料 §A 售后答)。

### E. 热销品类(选品型真实依据·与上方§4一致,强化)
没有绝对最好卖,**看当地市场需求**。欧洲采购最多=家居百货/厨房/收纳/家居装饰/汽车用品/手动五金等**日用百货**;南美除家居百货外,**电动工具/五金工具/美妆/服饰**需求大。**日用品=高频消费、复购最稳。**
