# 爆款 rubric 维度素材库

收集"什么样的文案会火"的可量化维度,供《内容教练闭环》的 rubric 起步与 bump 参考。
每条维度记:名字 / 含义 / 怎么打分(可观测信号)/ 来源。

> 注:rubric 是"工作台不是博物馆"——被数据推翻或已吸收为正式维度的观察要删,git history 才是档案。

---

## 增量价值 / 存在感（首条,2026-06-16）

**含义**:文案是否提供了"别人没有的东西"——新判断 / 新方法 / 亲身验证的数据 / 失败教训。
低创作 = 把网上已有内容换个表述复述一遍(原创度高 ≠ 增量高)。

**怎么打分(可观测信号)**:
- 是否含作者**亲身经历 / 失败 / 具体时间成本 / 产出对比**(有真实细节数字 → 高)。
- 是否有**作者的存在感**(读者感觉到真人在分享 vs 像百科条目)。
- 差异化是否=**同题不同切入**(别人讲道理→你讲经历;别人列清单→你展示过程;别人晒成功→你晒失败)。
- 是否命中**信息缺口**(评论区高频问题 / 现有内容没覆盖处)。
- 是否有**跨领域嫁接**(A 领域方法用到 B 领域)。

**反例信号(判低分)**:通篇番茄钟/时间块/清单法这类一搜到处都是的通用方法,无自身验证、无独家细节。
