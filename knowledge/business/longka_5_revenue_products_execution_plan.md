# Longka 5 个可收钱 AI 项目执行计划

目标：不是做一堆 AI demo，而是把现有技术手段整理成 5 个能卖、能交付、能复购的成品项目。

## 总原则

1. 先收尾色彩项目，证明 AI 能交付高质量结果。
2. 再把色彩图像素材变成内容获客能力。
3. 再做老板晨报，形成每日入口。
4. 再做销售跟进，接近成交环节。
5. 最后把这些能力装进“龙咖 AI 员工办公室 / U盘入职盘”。

产品语言统一为：AI员工，不讲 Agent、workflow、prompt、API。

---

## 项目 1：AI形象顾问员工

### 定位

客户上传照片，AI形象顾问生成试看图；客户满意后付款，生成 11 张高清形象报告图。

这是当前第一现金流项目，也是 AI 高质量视觉交付能力的样板间。

### 已有技术手段

- 43 服务器：个人形象报告生图工厂。
- 122 服务器：小程序、订单、支付、用户交付展示。
- gpt-image-2 / AIGOCODE 生图能力。
- Codex CLI 成熟出图路径。
- 彭元琴样片、overview-card-a3 等优质图参考。
- 小程序上传、试看、订单、支付、用户订单页雏形。
- 高清单图交付方向已经明确。

### 还缺什么

- 稳定禁止回退到 HTTP 524 老路径。
- 试看图质量稳定贴近满意样片。
- 管理员后台：看订单、看照片、看状态、手动重跑。
- 防滥用：免费试看次数、重复上传、失败重试规则。
- 风控：黄图、政治图、动物图、无效图拦截。
- 付款后 11 张图全套生产进度和失败补跑。
- 122 合法域名下展示成品图。

### 成品形态

- 小程序用户端。
- 管理员后台。
- 试看图页面。
- 付款页。
- 11 张高清成品图交付页。
- 单图放大、保存、分享。

### 收费方式

- 试看引导：9.9 元或活动价。
- 完整报告：99 / 199 / 399 元。
- B端合作：美业、形象顾问、穿搭博主、摄影机构。

### 近期任务

1. 43 生图路径固定为成熟 Codex CLI 路径。
2. 122 同步展示和状态流转稳定。
3. 管理员后台补齐。
4. 小妹完整测试：上传 -> 试看 -> 付款 -> 11 张图 -> 保存。
5. 沉淀 3-5 套优秀案例作为首页样片。

## 项目 2：AI内容运营员工

### 定位

帮老板每天产出朋友圈、小红书、视频号、短视频脚本、案例包装和活动文案。

它不是一个写文案工具，而是一个“每天交内容的运营员工”。

### 已有技术手段

- 色彩项目高质量图片素材。
- `longka-video-workbench-prototype`。
- `my-video`。
- Remotion / Hyperframes 视频生产能力。
- `LONGKA_VIDEO_WORKFLOW_MVP_PLAN.md`。
- 小红书/朋友圈/私域内容方法论文档。
- AI 图文、报告、海报、短视频脚本生成能力。

### 还缺什么

- 标准内容日历模板。
- 行业内容模板：美业、形象、门店、培训、私域。
- 从案例图自动生成短视频脚本和发布文案。
- 运营员工日报格式。
- 老板确认发布流程。

### 成品形态

- 内容运营工作台。
- 每日内容建议。
- 朋友圈文案 3 条。
- 小红书标题 + 正文 + 封面文案。
- 视频号短视频脚本。
- 30 秒成片/半成片素材。
- 老板确认后复制/下载。

### 收费方式

- 399-699/月：每日文案和选题。
- 999-1999/月：文案 + 图片/视频脚本 + 案例包装。
- 2999+/月：带人工复盘和行业配置。

### 近期任务

1. 用色彩项目做 3 条短视频样片。
2. 做一套“每日内容运营员工”输出模板。
3. 把色彩项目优秀案例自动包装成朋友圈/小红书内容。
4. 形成可演示页面：今天发什么、为什么发、怎么发。

## 项目 3：AI市场雷达 / 老板晨报

### 定位

每天早上老板打开 U盘/工作台，看到 AI 已整理好的行业情报、今日商机、同行动作、今日待办。

它是未来 AI员工办公室的第一屏，也是最高频入口。

### 已有技术手段

- `AI_Make_Money_2026.xlsx` 案例库分析能力。
- `ai_employee_boss_pain_matrix.md`。
- `LONGKA_PRIVATE_DOMAIN_INTELLIGENCE.md`。
- `INTELLIGENCE_PIPELINE_PLAYBOOK.md`。
- Agent Reach / 网页抓取 / 资料整理方法。
- 小红书、公众号、网页、社群信息分析经验。

### 还缺什么

- 每个行业的信息源配置。
- 情报筛选和评分规则。
- 从新闻到“老板今天该做什么”的转化模板。
- 晨报界面。
- 每日定时任务。

### 成品形态

老板晨报页面：

- 今日行业情报。
- 今日同行动作。
- 今日商机。
- 今日建议内容。
- 今日客户跟进提醒。
- 待老板确认。

### 收费方式

- 单行业晨报：199-399/月。
- 和 AI内容运营员工组合：999-1999/月。
- 行业定制情报：2999+/月。

### 近期任务

1. 先选一个行业：形象顾问/美业。
2. 建立 10-20 个信息源。
3. 每天生成一版老板晨报 Markdown。
4. 做成简单 Web 页面。
5. 接入内容运营员工：晨报里的商机能一键变成文案。

## 项目 4：AI销售跟进员工

### 定位

帮老板整理客户线索、识别成交机会、提醒跟进、生成回复话术。

第一阶段不做危险的个人微信自动发送，只做辅助决策和话术生成。

### 已有技术手段

- 私域情报方法论。
- wx-cli 曾验证过价值，但有封号/风控风险。
- 表单/后台/订单数据结构。
- 小程序订单和客户上传行为数据。
- LLM 摘要、分类、话术生成能力。

### 还缺什么

- 安全数据入口：手动导入、企微、飞书、表单、后台订单。
- 客户分层规则。
- 跟进优先级评分。
- 回复话术模板。
- 转人工和老板确认机制。

### 成品形态

- 今日待跟进客户。
- 高意向客户列表。
- 客户摘要。
- 推荐回复。
- 老客户唤醒话术。
- 跟进进度表。

### 收费方式

- 599-999/月：客户跟进提醒 + 话术。
- 999-1999/月：结合内容运营和老板晨报。
- 私域团队定制：2999+/月。

### 近期任务

1. 先从色彩项目订单数据做销售跟进。
2. 识别：上传未付款、看过试看未付款、付款未完成、老客户。
3. 生成跟进话术。
4. 管理员后台显示“今日该跟谁”。
5. 后续再扩展企微/飞书入口。

### 还缺什么

- 素材选择规则。
- 字幕排版和移动端检查。
- 批量生成能力。
- 输出封面图和发布文案。

### 成品形态

- 30 秒竖屏视频。
- 封面图。
- 标题。
- 发布文案。
- 六宫格故事板。

### 收费方式

- 单条 99-299 元。
- 10 条内容包 999-1999 元。
- 行业模板 U盘 / 月度代运营 2999+。

### 近期任务

3. 和 AI内容运营员工打通：每天推荐内容时可生成视频脚本。

## 最先卖什么

不要等 AI员工办公室全做完再卖。

销售顺序：

1. 先卖 AI形象顾问报告。
2. 再卖“帮你把案例变成朋友圈/小红书/视频号内容”。
3. 再卖“每天早上给你老板晨报”。
4. 再卖“客户跟进提醒和话术”。
5. 最后包装为 AI员工办公室月费。

## 当前第一优先级

继续色彩项目闭环，不开新战线。

但所有新代码和新页面都要为后面 AI员工办公室留接口：

- 用户上传 = AI员工接收任务。
- 试看图 = AI员工初步交付。
- 付款 = 任务升级。
- 11 张图 = 完整交付。
- 管理员后台 = 未来老板办公室后台雏形。
- 视频素材 = 内容运营员工素材库。

色彩项目不是孤立项目，它是 Longka 第一个可收费 AI员工案例。
