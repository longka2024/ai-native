# 小妹配图 / 漫画(Longka 专属 IP)

## 角色圣经(锁死,每张图都必须是同一个小妹)

**小妹** = Longka 自有卡通 IP,女性成长博主人设:

- **画风**:2D 扁平矢量插画(flat 2D vector illustration),干净线条、极简阴影。不是 3D、不是写实、不是厚涂。
- **长相**:年轻东亚女性,黑色低马尾、额前几缕碎发,圆润温和的脸,温暖的浅笑。**明确卡通化,非真人脸**(保护隐私 + 过平台)。
- **招牌穿着(夏季清凉装)**:珊瑚橘(coral / terracotta)短袖 T 恤 + 牛仔短裤 + 白色运动鞋。**珊瑚橘是她的招牌色**(品牌识别锚点),默认都穿这身。**注意季节**:现在是夏天,不穿羽绒服/厚外套;按内容季节可微调,但默认夏季清凉装。
- **气质/声音**:真诚、温暖、有同理心、不端着、不说教。像一个把自己也踩过的坑讲给你听的姐姐。
- **官方人设参考图**(出图时作为一致性参考,后端会以 image_urls 引用):
  - 全身基准:`media/persona/full_flat.jpg`
  - 表情:`media/persona/sheet/sheet_expr.jpg`(微笑/惊讶/思考/欢呼)
  - 姿势:`pose_point.jpg`(讲解指向)/ `pose_heart.jpg`(比心)/ `pose_facepalm.jpg`(扶额无奈)/ `pose_sign.jpg`(举空白牌可加字)

英文提示词里固定用这段角色描述,保证跨图一致:
> `2D flat vector illustration of the SAME recurring character "Xiaomei": a young East-Asian woman, black low ponytail with loose front strands, round gentle face, warm soft smile, wearing her signature coral/terracotta short-sleeve T-shirt, denim shorts and white sneakers (summer outfit). Clean flat style, simple lines, clearly cartoon (not a real face).`

## 核心方法(内化自小黑 2.0)

把文案的观点/处境,转译成一个**真实物件小现场**,让读者先看到真实、轻、怪的画面,再在 1 秒内「这说的就是我」:

```
小妹 + 一个真实物件 + 一个物理动作 + 2-4 个短中文标签 + 大量留白
```

铁律(违反即不合格):
1. **只一个核心物理动作 + 只一个真实主物件**(可配 1-2 小道具)。禁止把主题里的名词都堆进画面(元素清单化)。
2. **3 秒读懂**:不看说明就能 get 到冲突。需要解释才懂 = 失败。
3. **抽象状态用小妹的身体演**,别用字面实物比喻(表达"被压垮"就让小妹被一摞东西压住身体,不要画"一座大山写着压力")。禁止"(象征X)"注释。
4. **白底 / 近白棚**:标准配图纯白 `#FFFFFF` 摄影棚感;真实物件光影统一。
5. **短标签 ≤4(优先 3)**:手写感中文短词,不是标题、不是整句解释、不是假中文。
6. **禁**:截图、UI、Logo、PPT 信息图、箭头流程框、商业海报、大段文字。
7. **事实锚定(铁律3)**:人名/品牌/数字/经历只能用文案里给的,不编、不补全成更漂亮的履历;不确定的用概括标签("某平台""项目节点")。
8. **母版是质量标尺不是模板**:对齐留白/物件真实感/动作清晰度/标签密度,但禁止复刻物件组合和构图。每张至少 3 个变异点。

## 四种产出

### A. 配图模式(默认)
为文案出 3-6 张单张正文配图,每张讲一个物理冲突。小红书竖版 `3:4`/`2:3`,通用横版 `16:9`(看平台,默认竖版)。

### B. 漫画模式(知识/干货/有转折的故事用)
2-4 格连续分镜,**同一个小妹**用连续动作讲一个有钩子→冲突→转折/解法的小故事。每格一句短对白或旁白(气泡/短标签)。第 1 格 = 3 秒钩子。格与格之间靠小妹的动作和表情推进,不靠文字解释。

### C. 长卷故事模式(个人经历 / 成长复盘 / 成长路径)
超横版(`2.6:1`~`3:1`)的「真实物件人生线」:一条手绘曲线贯穿全图,串 5-8 个真实物件节点,每个节点小妹做一个动作 + 1-3 行手写短注;**左起**(起点身份/开场)→ **右收**(现在关注/结论)。节点沿曲线**自然不规律起伏**(不是水平平铺流程、不是规律正弦),节点间留白**不等距**、左右视觉平衡。高级**近白背景**(留极轻空气感,不死白)。节点文字像写在空白里的手写人生注释,**不用便利贴/卡片/序号/步骤标记**,彩色胶带/小圆点只做点缀。→ 女性成长「我的成长故事 / 逆袭复盘」最适合。

### D. 手绘逻辑图模式(拆流程 / 拆方法 / 拆结构 / 前后对比)—— 内化自小黑 1.0
把观点 / 流程 / 结构 / 判断 / 状态变化,转译成一张**手绘解释图**:清爽、怪诞但成立、可读但**不像说明书**。视觉 = 白底 + **黑色手绘线稿感 + 少量红/橙/蓝中文手写批注 + 大量留白**;**小妹承担核心动作**(不是站旁边装饰)。优先抓**认知锚点**:核心判断 / 两个断点 / 输入输出闭环 / 分流 / 前后对比 / 一鱼多吃 / 承接路径 / 常见坑 / 角色状态变化。**每次从当前文案重新发明一个"奇怪但成立"的隐喻,不复刻旧案例**。禁:PPT / 信息图 / 复杂架构 / 左上角类型标题 / 幼稚可爱。→ 女性成长/私校/美容的**干货拆解、流程、前后对比**最适合。
> 注意:这条只借小黑 1.0 的**手绘逻辑图方法**,角色仍是**小妹**(coral 夏装卡通)。要做**小黑土豆本尊**的图,用留底的 `ian-xiaohei-illustrations` / `ian-xiaohei-scenes`,别在这里混。

## 母版与质检(质量锚点)
- **母版 = 质量标尺,不是可复制构图**:出图前心里锁一张最接近的优秀构图当「比例/留白/动作清晰度/标签密度」的尺,但**至少改 3 处**(主物件/空间方向/小妹动作/道具/标签位置)。第一眼像旧图换字 = 失败。
- **硬性预算**:一个核心动作、一个主物件、≤1-2 小道具、≤4(优先 3)短标签;深色大物件(手机/电脑/黑盒)**不能支配画面**,要缩小或换更轻的物件;整体比母版**更轻或同等**,不能更满更重。
- **质检双闸(出图后过)**:
  - **CRITICAL(不过不交付)**:同一个小妹不漂移(对照 IP_DNA 锚点)/ 3 秒读懂 / 白底 / 短标签短少可读非假中文 / 无元素清单化 / 无母版换字 / 数字事实没编 / 最终图来自出图,不是 HTML/截图。
  - **STANDARD(可标注后迭代)**:留白与覆盖面对齐母版 / 小妹形体稳(不像表情包) / 真实物件不压画面 / 点缀色只做节奏。
- **视觉母版参考**:`external/ian-xiaohei-scenes/examples/`(小黑版 7 张)——**只当构图/留白/动作的质量锚点看,不复刻、绝不喂作角色参考**(角色永远用小妹 IP_DNA 的参考图)。**小妹专属母版图待生成(会花钱,以后报价)**。
