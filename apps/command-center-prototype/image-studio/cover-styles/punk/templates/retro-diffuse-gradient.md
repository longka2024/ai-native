一、核心目标
==================================================

请生成一张高级复古弥散风海报。

它必须同时满足以下标准：

1. 具有明显的复古弥散视觉
画面要有柔和扩散的色块、模糊光域、雾化边缘、胶片颗粒、印刷噪点、扫描质感、纸张纹理或低保真显影效果。

2. 不是固定模板
每次生成时，配色、弥散方向、纹理方式、留白比例、几何修饰、字体选择、文字排版都要自动变化。  
不能每次都是同一种颜色、同一种大圆、同一种标题位置、同一种左文右图结构。

3. 主标题必须清晰可读
无论标题大还是小，横排还是竖排，都必须保证主题词准确、醒目、无错别字。  
不能为了设计感牺牲可读性。

4. 画面必须有编辑设计秩序
可以极简，也可以强烈；可以高饱和，也可以低饱和；可以满版，也可以大留白。  
但整体必须有杂志排版感、艺术海报感和成熟视觉秩序。

5. 不要生成具体人物脸
除非用户明确要求人物，否则不要出现人脸、机器人脸、赛博人物、真实人像。  
优先使用纯颜色、弥散光域、抽象几何、模糊纹理、线条、圆形、矩形、波纹、文字暗纹等视觉元素。

==================================================
二、视觉随机化总规则
==================================================

请不要把这套风格理解成一个固定模板。  
它是一个“复古弥散视觉系统”。

每次生成都必须在以下维度中产生明显变化：

- 色彩方案
- 弥散角度
- 色块形状
- 留白比例
- 颗粒密度
- 字体风格
- 标题大小
- 文字方向
- 排版位置
- 几何修饰方式
- 印刷纹理方式

画面可以高饱和，也可以低饱和；  
可以是同色系，也可以是双色冲突；  
可以大标题压迫，也可以小标题留白；  
可以横向扫描，也可以纵向雾化；  
可以有强几何，也可以几乎没有几何。  

但所有结果必须统一保持：

复古弥散、印刷颗粒、抽象高级、编辑排版、标题可读。

三、色彩方案随机系统
==================================================

不要固定一种配色。  
请根据主题、用途和情绪，自动选择一种高级色彩方向。

可选色彩方向包括：

1. 高饱和强冲击
红 / 白 / 黑  
荧光绿 / 黑  
亮蓝 / 黄  
橙红 / 深蓝  
玫红 / 紫 / 黑  

适合：先锋、音乐、潮流、活动、强传播主题。

2. 中饱和复古胶片
橙 / 米白 / 焦糖  
蓝 / 灰 / 奶白  
红 / 粉 / 暗紫  
黄 / 绿 / 奶油白  

适合：杂志、生活方式、复古视觉、品牌封面。

3. 低饱和雾面艺术
灰绿 / 雾蓝 / 银白  
奶油黄 / 淡粉 / 灰橙  
暗青 / 黑蓝 / 雾白  
灰紫 / 米白 / 淡蓝  

适合：艺术展、知识封面、极简海报、设计作品集。

4. 单色系渐变
只使用一个主色系内部的深浅变化，例如：
蓝色系、绿色系、橙色系、粉色系、灰黑系、米黄色系。  
通过明暗、颗粒、雾化、纸纹和留白制造层次。

5. 双色冲突
只使用两个主色形成强烈关系，例如：
红白、蓝白、绿黑、橙蓝、粉黑、黄蓝、黑白。  
要求高级、有张力，不要电商感。

6. 意外实验配色
允许出现不常见但高级的组合，例如：
薄荷绿 + 黑字  
鲜红 + 雾白  
亮黄 + 深蓝  
淡紫 + 橘红  
青蓝 + 米灰  
玫粉 + 灰黑  
奶油白 + 荧光绿  
深蓝 + 暗橙  

要求：
颜色可以鲜艳，也可以克制。  
但必须有明确主色逻辑。  
不能变成廉价彩虹渐变。  
不能每次都生成同一套橙色或蓝色模板。

四、弥散结构随机系统
==================================================

弥散不是固定圆形光斑。  
请自动随机选择一种或多种弥散结构：

1. 横向扫描弥散
颜色像被横向拖拽、扫描、打印、复印，形成横向雾带、色带、速度感和旧显示器质感。

2. 纵向雾柱弥散
颜色从上到下形成竖向光柱、雾带、显影区域，像暗房显影或舞台光雾。

3. 中央留白弥散
画面中间保留大片白色或浅色空间，颜色从边缘向中间扩散，形成高级留白。

4. 边缘压迫弥散
强色块从四周向内压入，中间形成文字焦点，适合强视觉封面。

5. 大色块模糊弥散
几个巨大色块互相覆盖，边缘柔软，像失焦镜头、玻璃折射或热成像色域。

6. 波纹地形弥散
用柔和线条、地形线、波纹、流线表达扩散方向，形成抽象空间感。

7. 高噪点喷砂弥散
画面由大量颗粒组成，近看像印刷粉尘、喷枪、复印噪点，远看形成渐变。

8. 暗部显影弥散
深色背景中出现局部光域，像黑暗中显影的胶片、展厅灯光或夜间光雾。

9. 几何切割弥散
用半透明矩形、圆形、弧线、条带切割渐变，但几何必须轻盈、高级，不要像PPT装饰。

10. 纯色雾面弥散
几乎没有具体图形，只靠纸张纹理、颗粒、暗角、轻微色差和印刷感形成高级氛围。

11. 模糊光斑弥散
几个失焦光斑在画面中漂浮，但数量要克制，不能像廉价光效背景。

12. 低保真数字弥散
轻微压缩感、扫描失真、色带、颗粒、旧屏幕质感，但必须美观，不能脏乱。

要求：
每次生成时，弥散方向、角度、密度、模糊程度都要变化。  
不能固定为“左边文字 + 右边大圆 + 一条弧线”。  
色块边缘必须柔和，有空气感和旧印刷感。

五、排版随机系统
==================================================

文字排版必须多变。  
不能每次都是左侧大标题 + 右侧抽象图形。

请自动从以下排版方式中选择一种：

1. 巨型标题压迫式
主标题占据画面 50%–80%，字体极大，适合强传播封面。

2. 中央极简式
主标题较小，居中或竖排，依靠大面积弥散背景和留白制造高级感。

3. 边缘暗纹式
主标题居中或偏小，画面边缘出现巨大半透明英文或中文作为暗纹，不抢主题。

4. 杂志网格式
标题、副标题、英文、编号、小字形成严谨编辑网格，像独立杂志封面。

5. 纵向排版式
中文标题竖排，英文小字横排或边缘排布，形成东方艺术展海报感。

6. 横向分割式
文字沿横向白条、色带、扫描纹理排列，像艺术活动海报或实验品牌视觉。

7. 字体主导式
用字体本身成为画面主体，背景弥散作为情绪层。

8. 背景主导式
标题较小，主要让弥散色彩、颗粒和留白成为视觉主体。

9. 错位实验式
标题可以旋转、贴边、切边、局部压印、半透明重复，但必须可读。

10. 极简商业艺术式
只保留主标题和副标题，没有任何多余辅助文字，排版克制。

11. 满版杂志式
标题与画面边缘、角落、小字形成复杂但有秩序的编辑系统，适合艺术海报。

12. 海报九宫格灵感式
可以参考多张不同视觉海报的系统感：每次像同一视觉家族里的不同作品，而不是同一模板复刻。

要求：
每次生成的字体大小、位置、方向、字距、层级都要有变化。  
主标题必须准确可读。  
不要自动添加无关文字。  
不要让小字抢主标题。

六、字体随机系统
==================================================

不要所有图都使用同一种字体。  
请根据画面气质自动选择字体。

可选字体气质：

1. 现代无衬线粗体
适合强封面、音乐、潮流、科技主题。

2. 极细无衬线
适合艺术展、极简、低饱和、留白型画面。

3. 复古 Serif
适合杂志封面、电影感、艺术展、怀旧主题。

4. 压缩黑体
适合强标题、传播型封面、X封面。

5. 优雅宋体 / 明朝体
适合中文艺术感、展览、诗性主题。

6. 实验字体
允许轻微变形、拉伸、错位、压印、扫描模糊，但不能影响识别。

7. 中英混排字体对比
中文使用粗体、宋体、黑体或标题字体；英文使用细无衬线、Serif、压缩字体或打字机字体，形成编辑感。

8. 巨大边缘暗文字
可用极淡的大英文或中文作为背景暗纹，透明度低，只服务构图，不抢主标题。

要求：
字体必须和弥散背景发生关系。  
不能每次都是同一种大号中文字体。  
不能让标题变形到无法识别。  
中文不能错字，英文不能拼错。

七、信息密度控制
==================================================

请根据用途自动控制信息密度。

1. 如果用途是 X封面
只保留主标题、副标题，最多一处极小英文装饰。  
不要日期、地点、价格、二维码、嘉宾名单、活动排期。

2. 如果用途是公众号封面
中文标题必须清晰，副标题可以保留。  
小字尽量少，保证手机端能看清。

3. 如果用途是杂志封面
可以有少量栏目字、编号、英文标签，但必须克制、有网格秩序。

4. 如果用途是活动海报
只有用户明确要求活动海报时，才允许出现时间、地点、嘉宾、票价等信息。

5. 如果用途是艺术展海报
可以有少量日期、展览名、艺术家名、机构名式排版，但整体必须克制。

6. 如果用途是品牌视觉
文字更少，留白更多，弥散色彩更像视觉系统。

默认规则：
如果用户没有明确要求活动信息，禁止自动添加日期、时间、地点、票价、二维码、条形码、购票信息、嘉宾名单、Livehouse 信息、地址、电话。

九、材质与质感要求
==================================================

画面必须具有真实复古印刷质感。

请加入以下质感中的若干种：

1. 胶片颗粒
细密、自然、有层次，不要像粗糙滤镜。

2. 复古印刷噪点
像 Risograph、旧海报、复印机、胶印、杂志扫描。

3. 纸张肌理
轻微纸纹、布纹、磨砂表面、旧纸质感。

4. 半调网点
可以局部出现，增强印刷感，但不能让画面变脏。

5. 扫描痕迹
轻微扫描线、压缩感、复印感、色彩漂移。

6. 暗角和曝光漂移
边缘可以稍暗或轻微褪色，让画面有老胶片感。

7. 油墨扩散
文字或色块边缘可以有轻微油墨渗透感，但保持可读。

8. 低清晰度美学
画面可以略微模糊、雾化、失焦，但不能低质。

十、不同主题的自动策略
==================================================

### 如果主题偏 AI / 提示词 / 创作工具

避免机器人脸、赛博城市、蓝紫科技模板。

### 如果主题偏知识 / 方法论 / 教程
优先使用：
清晰标题、克制背景、杂志网格、索引感小字、留白、纸张质感。  
避免普通信息图和PPT模板。

### 如果主题偏女性 / 社群 / HerName
优先使用：
柔和但不幼稚的色彩、花瓣感弥散、丝绸感光域、细腻颗粒、优雅字体。  
避免廉价粉色、网红模板、人脸堆叠。

### 如果主题偏音乐 / 派对 / 乐队
优先使用：
高饱和或强对比弥散、扫描纹理、巨大标题、音乐海报排版、抽象声波线条。  
避免真实舞台照片、票务信息堆叠。

### 如果主题偏金融 / Web3 / 风险
优先使用：
深色显影、红蓝或绿黑冲突、数据感扫描、警示色暗流、强对比标题。  
避免直接堆K线、金币、比特币图标。

### 如果主题偏艺术展 / 文化 / 东方诗性
优先使用：
竖排标题、小字居中、边缘暗纹、大面积颗粒、雾化冷色、低调几何线条。  
避免过度商业化标题。

1. 主标题准确、清楚、可读。

3. 画面有胶片颗粒、印刷噪点、纸张纹理或扫描质感。
4. 配色每次可以不同，但都高级、有主色逻辑。
5. 弥散角度、形状、密度、留白比例每次都有变化。
6. 字体和排版每次可以不同，但都有编辑设计秩序。
7. 可以极简，也可以强烈，但不能模板化。
8. 像独立杂志封面、艺术展海报、实验音乐海报或高级品牌视觉。

10. 同一提示词多次生成时，应像同一个视觉系统下的不同作品，而不是同一张图换颜色。
