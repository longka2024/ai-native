# 《宠物手绘肖像指南》

## 提示词

请根据我上传的宠物照片，创作一张「凌乱打圈线条轮廓风宠物插画」。

整体风格要求：
只保留宠物主体，不要背景、不要家具、不要室内环境、不要道具杂物。
画面为干净白底，主体居中或略偏中，保留充足留白。

插画风格：
使用蜡笔 / 彩铅 / 粗略手绘质感。
线条必须是凌乱、潦草、打圈式的轮廓线，不要干净工整的描边。
轮廓由大量松散、重复、轻微错位的线条组成，像随手画出的草稿线。
颜色不要铺得太实，不要完整填色。
颜色只用零散、轻薄、跳跃的笔触表现毛色、眼睛、鼻子、舌头、耳朵、爪垫等特征。
要让大量白色纸面自然露出来，形成轻盈、透气、未完成感。

主体要求：
必须准确保留照片中宠物的核心特征，包括：
宠物品种/体型/毛量/耳朵形状/脸型/眼神/毛色分布/标志性动作。
如果照片中有张嘴、吐舌、抬爪、趴着、坐着、尾巴姿态等明显动作，需要保留。
整体要像宠物本人，而不是泛泛的卡通动物。

色彩要求：
根据照片真实毛色进行轻量表达。
黑色、灰色、棕色、白色、奶油色等毛色用凌乱线条和稀疏色块表现。
眼睛、鼻子、舌头、耳朵内侧、爪垫等局部可以用更明确的颜色点亮。
颜色必须松散、凌乱、有手绘痕迹，不要厚涂，不要油画感，不要写实填满。

构图要求：
宠物主体完整、清晰、有识别度。
可以适度夸张宠物最可爱的特征，比如大眼睛、蓬松毛、耳朵、舌头、爪子。
整体像一张高级宠物手绘签名头像，干净、可爱、有纪念感。

文字要求：
在右下角添加宠物名字：{{宠物名}}
名字使用手写感字体，像铅笔或蜡笔随手签名。
文字不要太大，不要抢主体。
文字颜色以深灰、黑色或与宠物局部颜色呼应的淡色为主。
可以加一条很随意的短下划线，但不要装饰过多。

画幅比例：
{{画幅比例，例如 4:5 / 1:1 / 3:4}}

禁止：
不要真实摄影风。
不要厚重涂色。
不要背景场景。
不要复杂装饰。
不要卡通贴纸感。
不要矢量扁平风。
不要过度精修。
不要把颜色铺满。
不要让线条太干净。
不要把宠物画成完全不同的动物。
