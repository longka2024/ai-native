# 语义转译极简

## 视觉风格原子

把输入的词、短句或概念转译成一张高级极简图形海报。文字不是说明标签，图像也不是独立插画；文字、主体、承载面、空间关系和少量色彩必须共同构成一个准确表达语义的视觉句子。

## 输入变量

- 主题词 / 主标题：{{主题词}}
- 核心文字 / 单词 / 词组 / 字母：{{核心文字}}
- 画幅比例：{{画幅比例}}
- 文字语言：{{中文 / 英文 / 中英混排}}
- 可选补充语境：{{补充背景，可留空}}
- 可选情绪倾向：{{情绪倾向，可留空}}
- 可选禁用元素：{{不想出现的元素，可留空}}
- 是否允许辅助文字：{{允许 / 不允许}}
- 辅助文字关系说明：{{辅助文字说明，可留空}}

## 风格基因

- 极简主场景，清晰承载面，1-3 个关键人物、物体或动作。
- 巨型核心文字成为画面骨架，像墙体、舞台、屏障、地形、容器或情绪压力面。
- 主体必须与文字发生关系：站在字前、嵌入字内、穿越字形、被字阻隔、依附承载面或借用负空间。
- 配色控制在 2-4 种主色关系内，纸张感、印刷感、版画感、轻微颗粒和克制纹理优先。
- 大面积留白可以存在，但留白必须服务语义张力。

## 图文关系

- 核心文字必须清晰可读，并成为第一视觉。
- 辅助文字只在被允许时出现，且必须直接深化主题，不能使用随机编号、假出版信息、无关坐标或装饰性句子。
- 若主题抽象，选择具体可见的对象和动作承载语义；若主题具体，通过尺度、距离、遮挡、对峙、给予、等待或穿越关系制造更深层含义。

## 风格特有禁用项

- 不要把词语当普通标题贴在插画上。
- 不要让图像和文字分离。
- 不要堆叠元素、复杂背景、廉价渐变或无意义装饰。
