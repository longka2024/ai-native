# 《怪诞灵魂手绘》《情绪放大镜》《灵魂小画》《一眼怪像》《手绘本真》

## 提示词

将上传照片转换成「怪诞手绘风」插画。

整体像彩色铅笔、蜡笔、圆珠笔混合画在米黄色纸张上的手绘作品。
不要数字插画感，不要AI精修感，不要动漫美型，不要写实修图。

核心目标：
保留主体辨识度，让熟人一眼认出是谁。
不要把人物画漂亮，重点是画出人物的性格、情绪和灵魂状态。

【人物处理】
根据原照片本身的表情和姿态进行夸张。
不要随机换表情。
开心的人更开心，严肃的人更严肃，发呆的人更发呆，思考的人更呆萌，高冷的人更高冷。
像把人物当下情绪放大 300%。

【眼睛硬性规则】
所有人物、猫、狗和小涂鸦角色的眼睛都必须是：
白色椭圆眼白 + 黑色小圆点眼珠。
禁止真实虹膜、真实瞳孔、美瞳、高光眼、动漫大眼。
眼白必须很多，黑点眼珠必须很小。
眼珠方向必须参与表情表达：
可以向上飘、向下飘、向左偷看、向右偷看、发呆、放空、无语、震惊、怀疑人生。

【嘴巴规则】
嘴巴要灵动，要比普通头像更有戏。
根据原照片情绪夸张嘴型。
可以出现：
微撅嘴、圆嘴、香肠嘴、抿嘴、张嘴发呆、偷笑、坏笑、傻笑、苦笑、露牙、吐舌头、嘴角抽动。
不要统一微笑。
不要恐怖畸形。
不要丑化人物。
嘴型要像情绪被放大，而不是随机搞怪。

【脸部和身体】
允许轻微改变脸型、额头、下巴、鼻子、耳朵、脖子、肩膀、头身比例。
可以头稍大、身体稍小、脖子缩短或拉长、肩膀松垮、姿态更别扭。
但必须保留原人物辨识度。
怪诞但可爱，夸张但不丑。

【宠物规则】
如果主体是猫、狗或其他宠物：
同样使用白色椭圆眼白 + 黑色小圆点眼珠。
保留毛色、耳朵、脸型、体型和原本表情。
可以让脑袋变大、身体变短、腿变细、耳朵放大、尾巴变长、毛发炸开。
根据宠物原本状态夸张成发呆、嫌弃、困惑、委屈、警觉、无语或呆萌。

【画风质感】
彩色铅笔上色。
蜡笔颗粒。
圆珠笔黑色线稿。
重复描边。
线条抖动。
颜色不均匀。
允许涂出界。
保留纸张纹理。
像真实手工画出来的手账插画。

【背景规则】
背景不要复杂写实。
优先提取原照片中的真实元素，转成手绘涂鸦。
也可以根据用户指定主题加入元素。
例如：
咖啡、电脑、书、本子、乐器、K线图、城市地标、宠物玩具、鱼、熊猫、小太阳、云、星星、音符、便利贴、手写吐槽文字。

如果背景太乱、太暗、没有价值，直接使用纯色背景：
米黄色、奶油色、浅灰色、浅蓝色、浅粉色均可。
背景必须服务主体，不要抢主体。

【构图】
主体是第一视觉中心。
周围元素像随手画出来的生活涂鸦。
画面可以有趣，但不要堆满无关元素。
简洁版只保留主体和少量指定元素。

【最终效果】
第一眼：很像本人。
第二眼：有点怪。
第三眼：很好笑，很传神。
整体真实、有趣、有性格，像把人物灵魂状态画出来。
