# SPEC — Longka 出图工作台(Image Studio)

> 2026-06-29 起草。SPEC-first:先对齐"做啥/不做啥/怎么验",用户拍板再动手。
> 顶层定位见记忆 `three-workbenches-architecture`(3工作台之②)。

## 1. 一句话目标
小妹在日常运营中,**所有出图需求**(小红书封面+内图、公众号封面+内图、抖音/视频号视频封面、海报、PPT图)——**选用途 → 给内容 → 看一排预览 → 选一个 → 出成品**,零技术、不挑引擎、不挑风格名。

## 2. 要解决的真问题
能力已经很足但**散落**:归藏 / 宝玉(封面/信息图/文章配图/知识卡)/ 小黑手绘 / 卷卷 / punk 12 / gpt-image 13,各是各的 skill,没有统一入口。小妹不知道该用哪个、参数怎么填。**缺的是"统一工作台 + 用途路由 + 多方案预览",不是新出图能力。**

## 3. 做啥(范围内)

### 3.1 按"用途"进,不按"风格名/skill名"进
入口 = 6 类运营用途(自动配比例 + 候选引擎/风格):
| 用途 | 比例 | 主力引擎 |
|---|---|---|
| 小红书封面 | 3:4 | 归藏卡 / 宝玉cover / punk / gpt-image |
| 小红书内图(轮播) | 3:4 | 归藏(28版式)/ 宝玉信息图(20×17)/ 手绘(小黑/卷卷/宝玉配图) |
| 公众号封面 | 21:9 + 1:1 对 | 归藏(专做这对) |
| 公众号内图 | 16:9 | 宝玉信息图 / 归藏 |
| 抖音视频封面 | 9:16 | 我们 Remotion封面 / 宝玉cover / punk |
| 视频号封面 | 16:9 / 竖 | 同上 |
| 海报 | 选比例 | punk 12 / 宝玉cover / gpt-image |
| PPT图 | 16:9 | 宝玉 slide-deck / 归藏PPT |

### 3.2 两种引擎,统一注册、清楚标注(别混)
- **HTML 渲染卡**(归藏 / 宝玉信息图·知识卡·slide-deck / 小黑·卷卷手绘):Playwright 截图。**免费、确定性、字不糊不乱、能放真实截图。**
- **AI 出图**(宝玉cover 5维 / punk 12 / gpt-image 13):**有冲击、但花钱(Kie/43)、字可能糊。** 花钱动作先报价等点头(铁律 [[2026-06-22-paid-action-consent]])。
- UI 不暴露 skill名/服务商,只说"卡片排版 / AI生成"两类。

### 3.3 统一风格库(注册表)
把现有全部内化进一张 `imageStyles` 注册表:每条 = { id, 名称, 用途适配[], 引擎(html/ai), 比例支持[], 后端route, 缩略示意 }。
- 来源:config.js 现 13 + 归藏(2体系×版式)+ 宝玉(cover/信息图/配图)+ 小黑 + 卷卷 + **punk 12 内化(剥作者引流)**。
- 去重/归并(punk商业杂志 vs 归藏 vs gpt-image poster 不重复堆)。

### 3.4 多方案预览(北极星 [[peitu-many-options-northstar]])
选用途 → 系统按内容推荐 N 个风格 → **一排真实缩略图**(不是给名字让她猜)→ 点一个 → 出成品。

### 3.5 统一质检门
出图后自动过:文对题 / 清晰可读 / 不溢出不糊(vision-judge GLM-5V + HTML validator)。AI图重出花钱留用户点,判图免费自动。

### 3.6 产品化
能力沉后端工作流 + 前端按钮;成品进"作品记录"(复用现有保存)。小妹零技术。

## 4. 不做啥(范围外,防膨胀)
- 不重建采集/选题/语料(吃**共享中台**,别按工作台重造)。
- 不做视频成片(那是③视频工作台);本台只出**视频封面(静态图)**。
- 不做音乐。
- 不改已认可的前端模块拆分结构。
- 不在对话框手搓交付(铁律:能力进工作台)。

## 5. 怎么验(验收)
1. 小妹选"公众号封面" + 贴一篇文 → 出 21:9 + 1:1 配套对,字准、可读、不糊。
2. 选"小红书内图" + 贴要点 → 归藏/宝玉信息图一排预览 → 选一个 → 出 3:4 吃满画布的卡。
3. 选"海报" + 主题 → punk/AI 一排 → 选一个 → 出图(花钱前有报价确认)。
4. 选"手绘配图" → 小黑/卷卷/宝玉手绘出图。
5. 全程不出现 skill名/服务商;每张过质检显示通过/建议重出;成品进作品记录。
6. 122 上小妹能用(不只本地自证)。

## 6. 分期(先小后大)
- **P1**:HTML 卡主力打通——归藏(小红书内图+公众号封面对)+ 宝玉信息图 + 现有13接进统一入口 + 多方案预览骨架。**免费、最高频、见效快。**
- **P2**:AI 出图线——punk 12 内化 + 宝玉cover 5维 + 手绘(小黑/卷卷)接入;质检门统一;报价确认。
- **P3**:视频封面联动(与③视频工作台共享封面)+ 作品记录/复用打磨。
