## 1.5 视频编辑(用户 2026-06-22 补认知:统一归类)
"形象换我的 / 场景换我的 / 衣服换 / 加电影感 / 删杂物" —— **全部统一归类为「视频编辑(video-to-video)」**,不是从零重生成。这是 2026 新成熟的能力,直接编辑现有视频。

**调研结论(API 现状 + 价格,2026-06):**
| 工具 | 能力 | API | 价格 | 备注 |
|---|---|---|---|---|
| **Runway Aleph 2.0** | change/replace/swap/add/remove/**restyle(电影感)**/relight,局部精确编辑 | ✅ `runway:aleph@2.0`(Runway / Runware) | **$0.05/秒**(~¥0.36/s,15s≈¥5.4),或~$1.5/条;输入 2-30s,≤1080p | **最全能**,首选云 |
| **Google Gemini Omni**(I/O 2026) | 换背景/删物体/改衣服/电影感/**保人脸保声音** | ✅ Gemini API(配合 Veo:Veo生成、Omni编辑) | 按时长计费 | 能力强,但**Google API 中国访问不稳**,慎用 |
| **Pika**(Pikaswaps/Pikadditions) | swap 衣服/背景/物体、加物体 | ✅ | **$0.20/5s(720p)、~$0.40/5s(1080p)** | **最便宜**,swap 类够用 |
| **ComfyUI 自建**(终局) | 换背景(SAM+video inpaint)/换装(video try-on)/relight | 自建 | **零按次** | 对齐北极星,等 GPU 就位 |

**红线(合规):视频编辑只对「我们自己拍的素材」用**(小妹自拍 → 换背景=我的场景、改衣服、加电影感、删杂物 → 提质感)。**绝不直接编辑+重发别人的爆款视频**(=搬运/侵权,平台判搬运、封号)。复刻 = 复刻结构/创意,再把视频编辑用在自己素材上。

**路线**:先按需用 **Pika(便宜 swap)/ Runway Aleph(全能)**;**终局 ComfyUI 自建零按次**。任何一次都先报价、经同意([[2026-06-22-paid-action-consent]])。

## 2. 确定性优先原则(落到每一环)
- **固定/可模板化的 = 用脚本/工作流跑**:字幕、字体特效、排版、拼接、卡点、合规改写、选题筛选、配图风格匹配 —— 全确定性,可复现,零随机。
- **必须生成的(图/视频/语音)= 抽卡区**:优先 ① 自建 ComfyUI(零按次)② 便宜云(Kie $0.03图)③ 固定素材库复用。**抽卡前必报价、必经用户同意**([[paid-action-consent]])。
- **ComfyUI 作为自建抽卡引擎**:出图(SDXL/Flux)、图生视频、口型数字人、声音克隆,全部跑自己 GPU/端脑云,边际成本≈0,工作流固定=确定性强。

## 3. ComfyUI 一条龙(终局架构)
```
确定性工作流(脚本)编排 →
  [需要抽卡时] 调 ComfyUI 自建节点:
     出图(Flux/SDXL)/ 图生视频(Wan/LTX)/ 数字人口型(LatentSync/EchoMimic)/ 声音克隆(CosyVoice/GPT-SoVITS)
  [固定部分] 脚本直接做:字幕/特效/拼接/合规/排版
→ 成片
```
