# SPEC — 程序化动态视频（重点大字为主 · A-roll/B-roll 镜头语言 · 点睛画面为辅）

> 状态：方向已定（用户选「程序化动态视频」），写 spec → 实现
> 日期：2026-06-18
> 取代：口播片里"AI 图文卡 → 图生视频摇晃"那条（被否：中文错别字、画面塞字、静图摇晃=敷衍）
> 关联：`2026-06-18-oral-broadcast-video-spec.md`（配音/数字人北极星仍有效，本 spec 是其"主体画面"的正确做法）

## 为什么换（根因）
- 把整段文字画进 AI 图 → 中文必出错别字、画面又密又乱、丑。
- 几张静图来回摇 → 廉价、敷衍。
- 用户判断：视频要创新；**每镜语言抓重点，画面只点睛，不堆一堆**；要有 **A-roll/B-roll 镜头语言**。

## 核心形态
**字是代码渲染的（零错别字、有设计、真动），画面只做点睛。** 每镜：
- **A-roll（主）**：这一镜的**一句重点** → kinetic 动态大字（关键词跳色/缩放/进出），配音读它；（后期）可叠**数字人**口播。
- **B-roll（辅·点睛）**：支撑这句的**简洁视觉** —— 大数字 / 图标 / 极简数据动画 / 一张干净配图 / 几秒真动空镜。**一镜最多一个点睛元素，不堆。**
- **镜头语言**：A-roll↔B-roll 交替、景别变化、卡配音节奏剪、干净转场；避免单一画面摇到尾。

### 一镜示例（你那条 AI 民调脚本·第1镜）
- A-roll：屏幕炸出大「**63%**」+ 副标「美国人慌了 · AI 跑太快」，关键词跳色。
- B-roll：角落一个极简"上升曲线/人群"图标轻动。
- 配音读「63%美国人慌了——AI跑太快」。
- 不是塞满便签清单的卷卷图。

## 引擎
- **主：Remotion（代码渲染视频，React/TSX）** —— 动态文字、数据动画、转场、合成 A-roll/B-roll + 字幕 + 音轨，headless Chromium 渲 MP4。零错别字、风格统一、可模板化批量（内容工厂要的）。
- **轻量回退：ffmpeg drawtext/zoompan** —— Remotion 没装好前先出"动态大字 + 点睛图"的简版。
- 122 已有 node + ffmpeg；Remotion 需装 `@remotion/*` + chromium（参考 remotion-video skill）。

## 五条件
### 范围
脚本 → 程序化竖屏短视频：每镜重点大字(A-roll) + 点睛画面(B-roll) + 配音 + 字幕；模板化、可批量、可配主题色/字体/品牌。
### 输入/输出
- 输入：确认文案（→归并成 N 镜，每镜抽一句"重点"+一个"点睛意图"）、主题色/风格、配音音色、(可选)真实参考图、(后期)数字人。
- 输出：MP4(9:16)，长度=配音；附每镜结构（重点句/点睛元素/时长）。
### 约束（铁律）
- **脚本 > 大模型**：文字/动画/数据/转场/合成全代码，不靠 AI 画图。
- **能不花钱不花钱**：A-roll 全免费（代码渲染）；B-roll 优先免费（图标库/代码数据动画/真实参考图），AI 真动空镜只在必要处、少量。
- **稳定 > 花哨**：122 本地渲染，失败如实报。
- **UI 不露黑话**；成本/风格/字体可配（一处）。
### 验收（反敷衍硬线）
- [ ] 每镜=一句重点大字(A-roll)，**画面不塞整段文字、无错别字**。
- [ ] 全片有 **A-roll/B-roll 交替**的镜头语言，不是单画面摇到尾。
- [ ] 关键词有动效/高亮；卡配音节奏；转场干净。
- [ ] 风格统一（主题色/字体锁定）；长度=配音±2s。
- [ ] 成本：A-roll 0 元；整条≤目标（默认远低于 AI 图卡方案）。
### 非目标（本期）
- 复杂三维/粒子特效；多轨混音。
- 数字人合成（P3，端脑云 HeyGem，叠加进 A-roll，后做）。

## 脚本→镜头 的"提示词元素"（生产时必须带）
- **每镜重点(A-roll 文案)**：从该镜脚本抽 ≤1 句"金句/重点"，+ 1-2 个高亮关键词。
- **点睛意图(B-roll brief)**：一句话说"这镜配什么点睛"（大数字/图标/数据/空镜）。
- **B-roll 视觉生成提示词(若用 AI/取图)**：`cinematic B-roll cutaway, single clean subject, shallow depth of field, subtle slow camera move, minimal, NO text/labels/words, matches: <该镜重点>`。
- **镜头语言标注**：景别(特写/中景/空镜)、入场动效、转场、停留时长(=该镜配音长)。

## 实现分步
1. **P1 脚本结构化**：归并成 N 镜，每镜抽 {重点句, 关键词[], 点睛意图, 景别}。先脚本规则，必要时轻量 LLM 抽取。
2. **P2 Remotion 模板**：竖屏动态大字模板(关键词高亮/进出动效) + 点睛槽位(图标/大数字/图片) + 转场 + 字幕 + 音轨；主题色/字体可配。
3. **P3 配音对齐**：复用已做的 MiniMax TTS（每镜配音→该镜时长）。
4. **P4 渲染管线**：122 装 Remotion+chromium；`/api/programmatic-video/start|status` 异步渲 MP4。
5. **P5 B-roll 资产**：免费图标/数据动画优先；AI 真动空镜按需(用上面提示词，少量)。
6. **P6 数字人叠加**（后，端脑云 HeyGem）。
