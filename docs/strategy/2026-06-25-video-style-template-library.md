## 一、通用口播骨架(所有模板公共 · Remotion 实现基座)
1. **画中画口播**:主画面铺满 + 形象窗(真人照/卡通,右下角或底部居中,圆角描边,留"可替换"位)
2. **跟读字幕**:居中偏下,粗黑体白字 + 描边/底衬,严格跟配音节奏(= 声画同步铁律 [[video-audio-sync-mandatory]])
3. **顶部章节导航**(可选):横向分段小标题(如 骶髂关节 | 不舒服原因 | 训练),显示"讲到哪一节"
4. **大字序号 / 点睛标题**:超大 01 / 02 分点(白色巨数);或 grunge 做旧大标题点睛
5. **数据可视化**(可选):彩色条形图(红/蓝/黄)+ 大字数字 + "数据来源"标注 + "替换素材"占位
6. **氛围层**:网点半调背景 / 粒子光束 / 景深推进 / 角落水印

### 档1 · 商业知识口播 → AI / mizan / 财经 【口播·画中画】
> ⭐ Gemini 精分析**正版小lin说·SpaceX IPO**(YouTube wpb-DrbhEiY,2026-06-25)——真爆款手法(比拆解复刻版真):
- **真人女主播出镜**(绿幕抠像+圆形画中画灵活切,全屏/角落切换,语速快+手势多)——画中画的人**理想是真人抠像**,AI形象/数字人只是没真人时的替代
- **钩子=语不惊人死不休**:开头甩最炸的结论+反差对比("人类史上最大IPO"+马云敲钟),制造信息落差强留观众
- **结构**:模块化推进(引言→三大板块→深度反转→核心机密→展望),节奏极快**每3-5秒一次画面切换/视觉更新**
- **招牌手法**:**原文档(PDF/截图)实时荧光高亮**(黄/粉划重点)+ 黑金/黑银科技感 UI 信息卡(展示关键数字对比)+ 全屏大字段落标题
- **视听同步**(高级感命门):音效卡点与数据跳动、切图严丝合缝;BGM 科技电子乐+悬疑管弦,音效用翻页/打字/叮/meme梗
- 字幕:居中底部粗黑无衬线,白字+关键词(金额/年份/术语)黄绿加粗,逐句跳出
- 配色:赛博朋克科技感,深色底+霓虹点缀,冷色温
- B-roll:实拍(火箭/演讲/交易厅)+ 截图(招股书/公告)+ 动效(3D渲染/财务曲线/柱状图/AI插画)

> 早期从拆解复刻版抽帧的版本(可替换占位/AI形象代替),保留作对照: Gemini 精分析「小lin说·香港vs新加坡」(拆解版)细化:
- **结构(铁律)**:钩子(对比悬念)→ 举例展开 → 新闻杂志风收尾
- **钩子公式**:**「X 和 Y,谁才是 Z?」** 对比悬念,前3秒抓人(如"香港和新加坡,谁才是亚洲金融一哥?")
- **A-roll = AI 动漫形象主持(会微动)+ AI 西装男快闪** —— **不是静态照片**(静态图=不够性感的根因);形象窗右下、标"可替换"。我们的对应:小妹卡通形象要**动起来**(数字人/微动),不是静态 jpg
- **B-roll**:城市天际线航拍(上海外滩等)+ 街景 + stock 拼贴(标"可替换")+ 玻璃信息卡(产品抠图+中英标题+¥价格)
- **收尾**:新闻杂志风格卡("时事快报"版式)
- 字幕:居中白字黑边(跟读);配色冷调高级 + 暖点缀;中速、信息密度高
- = mizan 混剪的**升级版**(加会动的画中画 + 杂志收尾 + 对比悬念钩子)

### 档2 · 数据硬核 → 数据 / 榜单 / 对比
- 参考:finance 条形图("三大外卖平台单日单量" 红蓝黄条 + 2000万/5000万)

- **⚠️ 适用前提(2026-06-25 教训·别套档硬凑)**:这档要"**通篇真实数据**"才配。mizan 真实数据只 3 组(到货/SKU/起订),硬做"数据完整版"结果只 1 个条形图+8 拍普通混剪=糊弄,再多就得重复/编数(违铁律3)。**已决定 mizan 不走此档**,走档0混剪/档1口播。数据硬核留给真有数据的榜单/财经对比。

## 四、Remotion 落地
每个风格档 = EffectEngine 的一组参数集(B-roll 类型 / 形象窗位置 / 字幕样式 / 序号组件 / 数据图组件 / 配色 / 特效 / 节奏)。流程:
**内容 → router 判型选档 → 取该档参数 → matcher 配 A-roll/B-roll → 声画同步配音 → Remotion 渲染**。高质精修走 VectCut(剪映桌面端·人在环 [[2026-06-22-digital-human-and-vectcut]])。
