# mizan 视频 SEO 关键词策略(2026-06-24)

## 目标客群
在**欧洲/南美开商超·百货·批发的华人老板**。看到 mizan 模式后,用 mizan 软件从**中国站下单采购**。只发**国内短视频平台(抖音/视频号/小红书)**,中文配音,文案**接地气**(客群文化水平不高)。

## 核心洞察:别人到底搜什么(最重要)
客户**不搜**"商超采购平台 / 供应链"(我们的内部词),他们搜的是真实口语意图词。证据(网络搜索 2026-06-24):

| 类别 | 真实搜索词 | 流量判断 | 证据 |
|---|---|---|---|
| **业态大词**(老流量盘) | 西班牙百元店、1欧店、华人百元店、华人超市 | 极大(新闻/内容海量) | 腾讯/人民日报/海外网多篇 |
| **进货意图词**(最值钱·高转化) | **华人超市在哪进货**、[地名]进货群、货源怎么找不踩坑、在国外怎么开中国超市、[地名]开店进货 | 中·精准 | `douyin.com/search/美国华人超市在哪进货` 直接命中抖音搜索页;寻友谷"[地区]华人超市进货群" |
| **地名长尾** | 阿根廷华人超市、智利百货批发、米兰唐人街批发、西班牙开超市 | 小·竞争小好排 | 中新网/人民日报侨网 |

**趋势**:百元店在衰退,华人转**超市/专业店**(美容/饰品/科技)。→ 关键词两头吃:"百元店"(吃老流量)+"开超市/进货货源"(吃新趋势)。

## 内容结构:70% 干货 + 30% 营销(纯广告没人看)
搜"华人超市在哪进货"的人,真实疑问 = **货源在哪、怎么不踩坑、飞回国值不值**。
- **70% 干货**:实打实答这个问——海外开店进货的几条真实路子 + 各自的坑(① 飞回中国跑义乌/广州批发:货全但机票住宿压货累又贵 ② 本地华人进货群拼海运:便宜但起订量大、等船一个多月、品类少 ③ 找货代代采:省心但层层加价利润薄)。
- **30% 营销**:mizan = 第四条路、最省的那条(手机中国站下单、源头工厂直发、海外仓就近补货,不用飞不用拼群)。结尾才落,搜进货的人先得到有用答案才愿看完。

## 选词 → 视频矩阵(一词一片,天然各不同 + SEO 覆盖)
每条片绑一个真实搜索词 + 一个地区,地名织进**标题/钩子/海外仓/收尾**(自然不堆砌):
- 华人超市在哪进货(总纲,西班牙站打样)
- 西班牙百元店 货源 / 意大利百货 进货 / 阿根廷华人超市 进货 / 智利百货批发
- 在国外怎么开中国超市 / 海外开店货源怎么找不踩坑
- 地名长尾各站(西/葡/意/法/德/阿/智/墨/巴)

## 平台 SEO(光视频里说没用)
标题 + 标签也要带词:标题=真实搜索词(如"西班牙开店进货,货源到底在哪进?");标签 #西班牙百元店 #华人超市进货 #海外开店货源 #mizan。

## 量化下一步(真平台热度)
网络侧只能定性。真热度排序用 **MediaCrawlerPro** 去抖音/小红书搜这些词、数**笔记量/点赞量**(高危搜索·退出保号),得出哪个词笔记多互动高 → 优先做。

## 这套方法要产品化(沉淀进工具,别手工搜)
"搜关键词 → 摸搜索热度 → 定选词 → 70/30 写文案"这套**工作方法本身**要做成系统能力(产品化北极星:能力沉后端工作流)。在管线里新增一层:

**关键词洞察层(SEO 选词器)** —— 输入(行业 + 目标地区列表)→ 自动:
1. **采热度**:MediaCrawlerPro 去抖音/小红书搜候选词、数笔记量/点赞量(高危·退出保号);TrendRadar/平台工具补充。
2. **排序选词**:按热度×转化意图×竞争度排,吐"该做哪些词、优先级"。
3. **生成 70/30 文案大纲**:每个选中词 → 真实疑问(70%干货:几条进货路子+坑)+ mizan 切入(30%),地名织进标题/钩子/海外仓/收尾。
4. 输出 = 一批"地区×搜索词"的文案节拍 + 平台标题/标签 → 喂下游匹配器/配音/渲染。

放进 spec 阶段(本期先手工跑通这套方法验证有效,再沉淀成 `seo_keyword_miner` 工具)。客户端只看到"系统按你的行业自动找对的选题和词",不暴露内部采集/排序逻辑。

## 落进管线
每条片 = 一个"地区×搜索词"的 70/30 文案 → 过文案质量门(dbs-ai-check/humanizer 保证接地气不堆砌)→ 匹配器配镜头 → IndexTTS 中文配音 → 渲染。关联 spec: `docs/specs/2026-06-24-content-factory-pipeline-spec.md`。
